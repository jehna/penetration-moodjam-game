Penetration

A game by Jesse Luoto.

Penetration is a game built fully in 48 hours for the first Mood Game Jam in Kouvola.

Gameplay

Protect your planet by clicking on the target you want to shoot at.

You can play the game in any browser or modern handlehed device.

The game is free-to-play, but good feedback is always encouraged at the main site.
